# ASTRVO OVERVIEW

**Astrvo** is an immersive 3D virtual event platform designed specifically for the Cardano community. It provides standardized or fully customizable virtual spaces for brand building, community gatherings, project demos, educational workshops, technical training, and NFT exhibitions. Users can log in with their wallets and participate through personalized avatars, visually showcasing their on-chain identity and assets while interacting with others and performing on-chain actions.
