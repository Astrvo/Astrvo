# Astrvo WebGL 模板

## 特点

这是一个极简的Unity WebGL模板，专为Astrvo项目设计，具有以下特点：

### 🎯 极简设计
- 无边框、无装饰的纯净界面
- 游戏内容占据整个浏览器窗口
- 响应式设计，适配各种屏幕尺寸

### 🚀 全屏体验
- 默认全屏显示，提供沉浸式游戏体验
- 支持F11键切换全屏模式
- 移动设备优化，自动适配屏幕尺寸

### ⚡ 快速加载
- 简洁的加载界面，显示进度条
- 使用Astrvo品牌logo作为加载图标
- 渐变进度条设计，视觉效果现代

### 📱 跨平台兼容
- 支持桌面和移动设备
- 自动检测设备类型并优化显示
- 响应式布局，适配不同分辨率

### 🎨 视觉设计
- 黑色背景，突出游戏内容
- 蓝色渐变进度条
- 简洁的警告和错误提示

## 文件结构

```
Astrvo/
├── index.html              # 主HTML文件
├── thumbnail.png           # 模板缩略图
├── README.md              # 说明文档
└── TemplateData/
    ├── style.css          # 样式文件
    ├── UnitySetup.js      # Unity初始化脚本
    ├── logo.png           # 加载界面logo
    └── favicon.ico        # 网站图标
```

## 使用方法

1. 在Unity中选择此模板进行WebGL构建
2. 构建完成后，游戏将自动全屏显示
3. 支持F11键切换全屏模式
4. 移动设备上自动适配屏幕尺寸

## 自定义

- 修改`style.css`中的颜色和样式
- 替换`logo.png`为自定义logo
- 在`UnitySetup.js`中添加自定义功能
