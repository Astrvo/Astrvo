# Roadmap

#### Milestone 1: Core Tech Stack & Wallet Integration&#x20;

Goal: Establish the technical foundation and connect user identity to the Cardano chain.\
Deliverables:

* Unity WebGL base project framework
* Successful integration of Mirror/Fishnet for multiplayer sync
* PlayFab user database setup
* Basic CIP-30 wallet connection plugin supporting login and address retrieval

#### Milestone 2: Virtual Space Construction & Avatar Integration

Goal: Develop the main virtual space and user avatars.\
Deliverables:

* Complete 3D modeling, texturing, and environment setup for the Snek-themed space
* Integrate Ready Player Me SDK for avatar loading and basic animations
* Users can freely walk and run inside the environment

#### Milestone 3: Multiplayer Interaction & NFT Display&#x20;

Goal: Deliver core social and NFT functionalities.\
Deliverables:

* Stable real-time text chat
* NFT display wall that shows selected NFT collections upon login
* Network optimization to ensure < 200ms latency for 50 concurrent users

#### Milestone 4: Deployment, Testing & Data Reporting&#x20;

Goal: Validate the MVP via a real community event.\
Deliverables:

* Online deployment for the Snek Taiwan meetup, including full technical support
* Activity data collection and a comprehensive user experience & performance report
* Open-source core space template and wallet integration plugin on GitHub

### Deliverables

* Accessible online 3D environment: A browser-based virtual space (Snek Den Taiwan example)
* Complete MVP functionality: Wallet login, avatar customization, multiplayer roaming, text chat
* Event deployment report: Includes participation numbers, session duration, and feedback
